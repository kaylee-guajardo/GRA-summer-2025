 # Contents of /Data for Kaylee/
 - call_data_cleaned_2020.Rdata - Contains R object "call_data_cleaned_2020". Each call is a row. See call_data_cleaned_2020.README for information about variables
 - 2020_site_habitat.csv - Rows correspond to individual sites. I do not have metadata about the variables contained. SITEID variable is the key to link with calls. In the call data it correspondes with SiteID.
 - 2020_cell_elevation - Rows correspond to gridcells. Variables indicate elevation summary statistics for elevation by grid cell. NABAT_CellNum is the key to link with calls.
